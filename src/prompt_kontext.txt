Проект и его цели:

Проект представляет собой прототип (MVP) игровой логики с клиентской и серверной частью.
Цель — создать онлайн-игру с режимами PvE и PvP, с поддержкой матчмейкинга, инстансов, системы прогрессии, ресурсов, боёв с монстрами и другими игроками, а также с возможностью получить артефакт и выйти через портал.
Предполагается пошаговая реализация за длительный период (180 дней), с учётом того, что автор — начинающий программист, выделяющий 2-5 часов в день.
Конечная цель — развёрнутый рабочий прототип с авторизацией, бэкендом, базой данных (PostgreSQL + Prisma), фронтендом на Next.js, деплоем на бесплатные сервисы (Vercel, Railway/Render, Neon.tech) и тестированием с друзьями.
Клиентская часть (Frontend):

Используется Next.js (React, TypeScript).
Структура проекта (по предоставленному скриншоту и коду):
src/app/ содержит:
GamePage.tsx — страница игры.
globals.css — глобальные стили.
layout.tsx — общий макет приложения.
ModeSelectionPage.tsx — страница выбора режима игры (PvE/PvP).
page.tsx — корневая страница (видимо, главная).
components/ содержит множество компонентов для отрисовки игры:
GameContext.tsx — контекст игры, хранит состояние игроков, карты, ресурсов.
GameManager.tsx — управляет логикой игры, подключает карту, инвентарь, обрабатывает действия игроков.
Inventory.tsx — компонент инвентаря, отображает и использует предметы.
Map.tsx — отрисовка карты, видимых тайлов.
Player.tsx — отображение информации об одном игроке.
Players.tsx — отображение всех игроков.
Resource.tsx — компонент ресурса (с подсказками).
Tile.tsx — отображение отдельной клетки карты.
Tooltip.tsx — всплывающая подсказка.
resources/ResourceData.ts — данные о ресурсах (еда, дерево, вода, камень, железо).
logic/ содержит модули логики:
artifactLogic.ts — логика артефакта (подбор, потеря, уведомления).
battleSystem.ts — логика боя (атаки).
generateMap.ts — генерация карты с террейном, ресурсами, бочками, монстрами.
inputHandler.ts — обработка ввода с клавиатуры (движение, взаимодействие).
matchmaking.ts — (мок) логика подбора игроков и выдачи instanceId.
monsterData.ts — данные о монстрах (гоблины, орки, тролли).
monsters.ts — логика поведения монстров (атаки после хода).
progressionSystem.ts — логика завершения инстанса, начисления наград, опыта и валюты.
terrainData.ts — данные о террейне (земля, лес, горы, лёд, вода).
styles/ — папка со стилями (inventory.css, map.css, player.css, resource.css, tile.css, tooltip.css).
Серверная часть (Backend) и база данных:

Предложено использовать Node.js + Express или Next.js API Routes.
База данных: PostgreSQL через Prisma.
Аутентификация: JWT, bcrypt для хэширования паролей, подтверждение email (SendGrid/Mailgun).
Миграции через Prisma.
В будущем: деплой на бесплатные сервисы (Vercel — фронт, Render/Railway — бэк, Neon.tech — БД).
Основные задачи/пункты развития:

Изучение основ JavaScript/TypeScript, React, Next.js, Git.
Настройка окружения: ESLint, Prettier, Docker для БД.
Реализация регистрации, логина, подтверждения email.
Реализация матчмейкинга, создание инстансов (PvE/PvP).
Генерация карты, расстановка ресурсов, бочек, монстров, портала.
Боевые системы (атака монстров, игроков), сбор артефакта.
Финализация инстанса (выход с артефактом), начисление наград, прогрессия.
Улучшение UI/UX: инвентарь, подсказки, фильтры, отображение характеристик.
Добавление WebSocket (Socket.IO) для обновлений в реальном времени, PvP.
Прогрессия: опыт, уровни, внутриигровая валюта, магазин предметов.
Квесты, события, балансовые правки.
Деплой проекта, тестирование с друзьями, сбор обратной связи.
Подробный план на 180 дней, разбитый на этапы, где каждый день ~2-5 часов.
Цели проекта:

Создать полнофункциональный прототип игры, в которой игроки могут:
Регистрироваться и авторизовываться.
Выбирать режим игры и подключаться к инстансу.
Исследовать карту, собирать ресурсы, сражаться с монстрами.
Взаимодействовать с другими игроками (PvP).
Получить артефакт и вынести его через портал для победы.
Повышать уровень, улучшать характеристики, покупать предметы.
Достичь готовности MVP за ~180 дней при ограниченном времени на разработку.
Научиться технологиям: Next.js, React, Node.js, БД, аутентификация, деплой.
